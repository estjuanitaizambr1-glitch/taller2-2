Curvas de Bezier

Las curvas de Bézier son métodos matemáticos utilizados para definir formas suaves y paramétricas en gráficos por computadora y diseño, basadas en un conjunto de puntos de control. Creadas por el ingeniero francés Pierre Bézier en los años 60, estas curvas se utilizan en herramientas como Adobe Illustrator, CSS y diseño automotriz (Renault) para modelar trayectorias suaves y precisas.
